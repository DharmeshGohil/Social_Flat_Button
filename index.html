Social Flat Button in CSS

Flat, small and large size social media buttons using Bootstrap and Font Awesome. Oh, and they're free too! Social Flat Button for Twitter Bootstrap

View project onGitHub

Social_Flat_Button

Example usage

Using this code for Small Button

<button class="btn google-flat"><i class="fa fa-google-plus fa-2x"></i></button>

Using this code for large Button

<button class="btn twitter-flat large"><i class="fa fa-twitter fa-2x"></i></button>
Note: if you add facebook large boutton than use this code
<button class="btn facebook-flat large-facebook"><i class="fa fa-facebook fa-2x"></i></button>

Quick installation

  1. Download social-Flat-button.css and include in your project. Done!

Installation

    Full instructions can be found here, but basically:

  1. Navigate to the Github project.
  2. Use buttons!

Requirements

  1. Bootstrap 3.v or newer
  2. Font Awesome 4.0.3 or newer
